## 📋 User Flows

### Report Creation Flow
```
Landing → Login → Dashboard → New Report → 
Upload Photo → Add Location → Select Category → 
Add Description → Review → Submit → Success → 
View on Map
```

### Organization Onboarding Flow
```
Landing → Sign Up (Org) → Enter Business Info → 
Upload Documents → Payment Setup → 
Verification Pending → Admin Review → 
Approved → Dashboard → First Work Order
```

### Cleanup Workflow
```
Citizen Reports → Admin Reviews → Assigns to Org → 
Org Accepts → Team Assigned → Work Started → 
Photo Upload → Completion → Citizen Verification → 
Payment Released → Rating
```
